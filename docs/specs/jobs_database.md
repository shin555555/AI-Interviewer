# 職種データベース（40職種マスターデータ）

本データは要件定義書に基づき策定された、AIによる初期の「マッチング要求スコア」です。
**最終的には Googleスプレッドシート上の『職種マスター』シートに配置され、システムに動的に読み込まれます。**
これにより、施設職員が地域の求人要件に合わせて、エクセル感覚で要求スコアを簡単に書き換える（微調整する）ことが可能になります。

## 各項目の定義（1〜5の5段階評価）
* **P1**: 正確性 / **P2**: 持続力 / **P3**: 体力管理 / **P4**: IT道具 / **P5**: 報連相
* **I1**: 集中力 / **I2**: 感情制御 / **I3**: 規律性 / **I4**: 柔軟性 / **I5**: 自己発信
* ※ **I4 (柔軟性)** は適性評価時に反転計算されるため、ルーティンワーク（検品・データ入力など）ではあえて低い数値（1や2）が設定されています。
