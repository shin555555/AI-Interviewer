## 11. 大規模アップデート: マッチング精度とバリエーションの向上 (新規タスク)
- [x] 質問プールの拡充: 現在の30問から、各属性9問×10属性＝計90問へ質問のバリエーションを増やす。
- [x] ランダム出題ロジックの実装: テスト開始時に、全90問の中から各属性ごと均等に3問ずつ（計30問）をランダムに抽出し、出題する仕組みを作る。これにより回答の「慣れ」を防ぐ。
- [x] 選択肢の拡充（4〜5択化）: 現在の3択から4択〜5択（具体的な行動や頻度を示す選択肢）へ変更し、ユーザーの「色の濃淡（グラデーション）」をより正確にデータ化する。
- [x] スコアリング調整: 選択肢が増えることに伴い、マッチングアルゴリズムや職種マスター（要求スコア）の基準値を再集計・調整する。
