# AI ワーク・プロファイル (自己理解サポーター)

就労継続支援A型・B型の利用者が、AIとの心理的に安全な対話を通じて自己特性（強み・課題）を言語化し、就職活動に直結する「プロフェッショナル・ポートフォリオ」を作成するためのWebアプリケーションです。

## 主な機能

1. **AIインタビュー機能**
   - 30問のカモフラージュ質問で「やさしい日本語」を用いた対話型インタビューを行います。
   - インタビュー開始時に「お名前（またはID）」を入力することで、管理側のデータと安全に紐づけます。
   - いつでも中断でき、4桁の再開コードで続きから再開できるセッション保持機能（LocalStorage）を備えています。
2. **プロフェッショナル・ポートフォリオ生成**
   - 回答のメタデータ（回答時間、修正回数など）も含めて10項目の特性スコアを分析。
   - 本人の強みの言語化や、最適な職種カテゴリ・推奨職種（TOP5）、さらに企業への配慮事項（処方箋）を出力します。
3. **管理者ダッシュボード機能**（🔐 パスワード保護付き）
   - トップ画面から「管理者画面」へパスワード認証を経てアクセスできます。
   - スプレッドシート（Google Apps Script API）から取得した全利用者のデータを基に、事業所全体の適性傾向や課題の自動グラフ化、利用者の時系列成長記録や個別レポート詳細の確認、PDF出力などを一括で行えます。

## 環境構築と起動方法

本プロジェクトは [Vite](https://vitejs.dev/) + [React](https://react.dev/) で構築されています。

### 1. 依存関係のインストール

```bash
cd app
npm install
```

### 2. 環境変数の設定

`app` ディレクトリ内に `.env` ファイルを作成し、以下の変数を設定してください（`app/.env.example` を参考にしてください）。

```env
# 管理者ダッシュボードへアクセスするためのパスワード (未設定の場合は admin1234 となります)
VITE_ADMIN_PASSWORD="your_secure_password"

# [今後実装] Google Apps Script (GAS) 連携用APIエンドポイント
VITE_GAS_API_URL="https://script.google.com/macros/s/..."
```

### 3. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173/` にアクセスして動作を確認してください。
